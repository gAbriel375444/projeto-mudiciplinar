SOBRE NÓS

Olá! Este site é um projeto para a matéria multidisciplinar do curso de Desenvolvimento de Sistemas da Escola Estadual Exército Brasileiro, do ano de 2026.


O projeto consiste em um chatbot acolhedor, feito para ouvir você, entender o seu problema e aconselhar de forma breve — ou simplesmente alguém para conversar e trocar uma ideia.

Para despertar novos interesses no usuário, criamos a área de HOBBIES, com diversas opções. O usuário pode aprender crochê, a fazer origami ou até mesmo dar os primeiros passos no violão.
